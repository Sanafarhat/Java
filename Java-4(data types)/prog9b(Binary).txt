Binary  Integer
-------------------
1) What  is  the  prefix  of  binary  number ?  ---> 0B  (or)  0b

2) What  are  the  valid  digits  in  binary  number ? ---> 0  and  1

3) What  is  the  base  of  binary  integer ?  ---> 2  due  to  two  digits  0  and  1

4) Which  digits  are  not  permitted  in  binary  number ?  ---> 2  to  9

5) int  x = 0B10101;
    What  does  variable  'x'  contain ?  --->Decimal  equivalent  i.e.  16 + 4 + 1 = 21

6) JVM  automatically  converts  binary  number  to  decimal  number  which  is  assigned  to  the  variable

7) System . out . println(binary-number);
     What  is  printed ?  ---> The  decimal  equivalent

8) binary  int  x  = 0B10100;
    Is  the  statement  valid ?  ---> No  becoz  there  is  no  keyword  binary  in  java
