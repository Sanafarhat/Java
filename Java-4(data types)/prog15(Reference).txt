Reference
-------------
1) How  is  a  reference  created  ?  ---> classname  reference

2) int   x;
    What  is  'x' ?  ---> Variable  becoz  int  is  a  datatype

3) String  s;
    What  is  's'  ?   ---> Reference  becoz  String  is  a  class

4) Date  d;
    What  is  'd' ?  ---> Reference  becoz  Date  is  a  class

5) Integer  i;
    What  is  'i' ?  ---> Reference  becoz  Integer  is  a  class

6) What  does  reference  contain ? ---> Hash  code  of  the  object

7) Where  does  reference  reside ?  ---> In  stack  memory

Note:
1) How  is  a  variable  created ?  ---> datatype  variable
    How  is  an  object  created ?  ---> new  classname()
    How  is  a  reference  created ?  ---> classname  reference

2) What  does  a  variable  contain ?  ---> Value  i.e. single  value
    What  does  an  object  contain ?  ---> A  group  of  Values
    What  does  a  reference  contain ?  ---> Hash  code  of  the  object

3) Where  does  variable  reside ?  --->  In  stack  memory
     Where  does  object  reside ?  ---> In  heap  memory
     Where  does  reference  reside ?  ---> In  stack  memory

4) What  is  left  side  of  =  operator  when  right  side  is  a  value ?  --->  Variable
												                       Eg:  int  x  = 25;
    What  is  left  side  of  =  operator  when  right  side  is  an  object ?  --->  Reference
												                          Eg:  String  s  = new  String("Hyd")
